# Rim-Craft
> **v0.0.1 WIP**

## 📖 简介

Inspired by **Rimworld** ，将生存管理、科技发展等要素融入 Minecraft 世界。

## 🎮 核心玩法模块

### 🌍 **环境系统**
- 四季变化与温度系统
- 动态天气与自然灾害
- 地形多样性与高度决定的资源分布

### ⛏️ **采矿与冶金**
- 多层次矿物系统，不同高度出现不同矿种
- 从石器、铜器、青铜到铁器的工具进阶
- 温度与火焰管制的冶金工艺

### 🌾 **农业与畜牧**
- 季节影响的作物生长周期
- 复杂的动物繁殖与管理系统
- 农业与狩猎相结合的食物链

### 🛡️ **防御与威胁**
- 增强的敌对生物与周期性袭击
- 防御建筑、陷阱与军事建设

### 💠 **生存与文明**
- 深化的生存要素（饥饿、健康、体力）
- 陶瓷、玻璃等手工制作系统
- 从原始到高度文明的发展阶段

### 🔬 **科技进度**
- 多阶段科技树，每个时代有独特的建筑与工具
- 自动化与工业化的终极目标
- 复杂的生产链与物流系统
